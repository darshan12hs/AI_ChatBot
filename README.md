# LangChain Chatbot Experiments

This project contains experiments using LangChain to build different types of chatbots with large language models (LLMs).

## Overview

I explored how LLMs can be used through APIs and how concepts like prompt engineering, embeddings, and retrieval work in real applications.

## What I Built

- Basic question-answer chatbot using Hugging Face models  
- Conversational chatbot using OpenAI (ChatOpenAI)  
- Prompt-based control of chatbot behavior (tone, style, creativity)  
- Streamlit-based chatbot deployment  
- Document-based question answering using PDF + FAISS (RAG)  

## Key Concepts

- Prompt engineering  
- Vector embeddings  
- Semantic search  
- Cosine similarity  
- Retrieval-Augmented Generation (RAG)  

## Tech Stack

- Python  
- LangChain  
- OpenAI API  
- Hugging Face  
- FAISS  
- Streamlit
